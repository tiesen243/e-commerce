# TN Store

Full Stack E-Commerce Web Application with NestJS, NextJS, Redux, TypeScript, Shadcn UI, MongoDB, Firebase and deployed on Vercel

## Installation

```bash
git clone https://github.com/tiesen243/e-commerce.git
cd e-commerce
yarn install
```
